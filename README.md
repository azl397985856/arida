## Introduction
一个用于数据订阅的工具库，尤其适用于扩展react，以允许子组件可以订阅父组件的数据，当父组件的数据更新时，及时更新子组件数据。


## Install

```bash
npm install arida
```

## Usage example

``` api
import { api } from 'arida';
api.subcribe('name', callback);
```
